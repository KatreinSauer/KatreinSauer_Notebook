This posts shows how to easily create a new .md file.

### How to create a new .md file

**1) Go to your _post folder you want to Use as template** \
    &#8594; That coud be: TEMPLATE_TO_CREATE_MD_FILES \
    &#8594; click on Edit (pen top right) \
    &#8594; mark everything (Ctrl+A) and copy \


**2) Go back to _posts folder** \
    &#8594; Add file (top right) \
    &#8594; Create new file \
    &#8594; Rename.md (.md you must also add!) \
    &#8594; Paste content from template \
    &#8594; Commit \
    &#8594; Fetch locally \
